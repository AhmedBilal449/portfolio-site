# Public Assets

## Resume

To enable the resume download functionality:

1. Add your resume PDF file to this directory
2. Name it `resume.pdf`
3. The download button in the Hero section will automatically serve this file

The resume download button is located in the Hero section of the homepage and is linked to `/resume.pdf`.

## Other Static Assets

You can place other static assets like images, icons, or documents in this directory. They will be served from the root path in production.
